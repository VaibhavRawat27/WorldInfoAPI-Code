This API lets you fetch **current time, date, weather, cultural notes, public holidays, NASA image, and much more** for any country in the world.

Base URL: http://127.0.0.1:5000 (Replace with your deployed URL)

‚úÖ Endpoints

1) Get Time & Information

Method: GET

URL:

/time?country=<Country Name>

/time?timezone=<Timezone Name>

Example:

http://127.0.0.1:5000/time?country=Japan

Sample Response:

{ "country": "Japan", "flag_image": "https://flagcdn.com/w320/jp.png", "country_image": "https://source.unsplash.com/600x400/?Japan", "date": "2025-07-17", "day": "Thursday", "time": "14:32:11", "utc_offset": "+0900", "greeting": "Good Afternoon", "weather": { "temperature": "29.4¬∞C", "condition": "Clear Sky" } }

2) Compare Two Timezones

Method: GET

URL:

/compare?from=<Timezone1>&to=<Timezone2>

Example:

http://127.0.0.1:5000/compare?from=Asia/Kolkata&to=America/New_York

Sample Response:

{ "from": "Asia/Kolkata", "to": "America/New_York", "time_now_from": "18:22:12", "time_now_to": "08:52:12", "difference_hours": 9.5 }
